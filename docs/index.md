# Agent Patterns

Welcome to the Agent Patterns documentation!

Agent Patterns is a toolkit for building advanced LLM-based agents using proven patterns and best practices. This library provides implementations of various agent architectural patterns that have been proven effective in different applications.

## Features

- Modular implementation of popular agent patterns
- Flexible memory system for maintaining context and knowledge
- Extensible tool integration for external capabilities
- Utilities for building custom agents and workflows

## Getting Started

To get started with Agent Patterns, check out the [Getting Started Guide](guides/getting_started.md) and explore the different patterns available.
